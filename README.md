# Angular2Music

以 [Angular CLI](https://github.com/angular/angular-cli) version 1.3.1 开发.

## Development server

运行 ng serve，启动后访问 http://127.0.0.1:4200 查看.

## 功能
> 1、单曲搜索

> 2、精品歌单

> 3、收藏的歌单(按网易云音乐昵称查询,将改为登录)

> 4、播放

> 5、歌词
